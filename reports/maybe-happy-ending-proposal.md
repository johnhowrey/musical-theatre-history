1. **only original productions count** (revivals don't), and
2. **no islands** — a show only goes on the map if it *connects* to someone already there —

| Creator | Role | Other Broadway work |

Do nothing. Maybe Happy Ending isn't on the map because it doesn't connect. Clean and consistent with the rules; the cost is a notable absence.

**The catch to be aware of:** the stations he'd connect to (Once on This Island, Parade) are the **original** productions (Ahrens/Flaherty 1990; JRB 1998). Arden directed the **revivals**, not those originals. So the line asserts an Arden link to shows he *revived*, not *originated* — which is exactly the thing rule (1) is meant to exclude. Option 2 only works if you're comfortable treating "directed an acclaimed revival" as a real enough connection for recent shows.

**Routing sketch (if Option 2):** place Maybe Happy Ending near Once on This Island (Ahrens/Flaherty cluster, ~x1450–1550, y860–900 has room around Ragtime/Seussical), drop an Arden-colored line from it to the Once on This Island circle, octolinear (H/V/45° + curved turns).
